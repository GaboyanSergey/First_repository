В данном репозитории хранится проект "Калькулятор". Он создан по заданию в университете.
Проект выполнен на языке C.
Для работы калькулятора пользователю необходимо выбрать операцию из шести доступных, ввести два числа, после чего он получит новое значение

Доступные операции:
1)Сложение
2)Вычитание
3)Умножение
4)Деление
5)Факториал 
6)Возведение в степень

Также калькулятор работает с векторами. Для них доступно 3 операции: сложение, вычитание, скалярное произведение.
